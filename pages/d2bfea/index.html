# 104. 二叉树的最大深度

  • 题目：104. 二叉树的最大深度 (opens new window)

  • 讲解：文章讲解 (opens new window)、视频讲解 (opens new window)

  • 要求：理解深度和高度的区别

# 录前想法

层序遍历，记录最大层数

以下是代码实现，已 AC (✔️)

class Solution {
public:
    int maxDepth(TreeNode* root) {

        queue<TreeNode*> que;
        int depth = 0;

        if(root) que.push(root);

        while(!que.empty()){
            
            int size = que.size();

            while(size--){
                TreeNode* tmpNode = que.front();
                que.pop();

                if(tmpNode->left) que.push(tmpNode->left);
                if(tmpNode->right) que.push(tmpNode->right);
            }

            depth++;
        }

        return depth;
    }
};

# 录后想法

还是第一次了解深度和高度的区别，简单来说就是：

  • 深度：节点到根节点的距离， 从上到下，适合前序遍历（深度是往下，挖呀挖呀挖）
  • 高度：节点到叶子节点的距离，从下到上，适合后序遍历

那如何求最大深度呢？

也就是从根节点到叶子节点的距离，可以使用求解高度的后序遍历

代码实现如下：

class Solution {
public:
    int maxDepth(TreeNode* root) {

        if(root == nullptr){
            // 为什么返回的是0 ？
            // 因为
            return 0;
        }

        // 后序遍历
        int left = maxDepth(root->left);    // 左
        int right = maxDepth(root->right);  // 右
        int height = 1 + max(left,right);   // 中

        return height;

    }
};

  • 讲解：文章讲解 (opens new window)、视频讲解 (opens new window)

  • 要求：

# 录前想法

层序遍历

以下是代码实现，已 AC (✔️)

class Solution {
public:
    int maxDepth(Node* root) {
        
        queue<Node*> que;
        int height = 0;
        if(root) que.push(root);

        while(!que.empty()){

            int size = que.size();
            height++;

            while(size--){

                Node* tmpNode = que.front();
                que.pop();

                for(int i = 0; i < (tmpNode->children).size(); ++i){
                    que.push((tmpNode->children)[i]);
                }
            }
        }

        return height;

    }
};

# 录后想法

推广到 n 叉数，更加理解遍历在做什么，遍历记录每一个子树的高度

以下是实现代码，已 AC (✔️)

class Solution {
public:
    int maxDepth(Node* root) {

        if(root == nullptr) return 0;

        int depth = 0;
        for(int i = 0; i < (root->children).size(); ++i){
            // 对比每一个子树的高度
            depth = max(depth,maxDepth((root->children)[i]));
        }
        
        return depth + 1;
    }
};

# 111. 二叉树的最小深度

  • 题目：111. 二叉树的最小深度 (opens new window)

  • 讲解：文章讲解 (opens new window)、视频讲解 (opens new window)

  • 要求：

# 录前想法

层序遍历，当遇到第一个左右节点都为空的节点（叶子节点），就是最小的深度

以下是代码实现，已 AC (✔️)

class Solution {
public:
    int minDepth(TreeNode* root) {

        queue<TreeNode*> que;
        int depth = 0;

        if(root) que.push(root);

        while(!que.empty()){

            int size = que.size();
            depth++;

            while(size--){

                TreeNode* tmpNode = que.front();
                que.pop();

                if(tmpNode->left) que.push(tmpNode->left);
                if(tmpNode->right) que.push(tmpNode->right); 
                // 遇到叶子节点
                if(tmpNode->right == nullptr && tmpNode->left == nullptr) return depth;
            }

        }

        return depth;

    }
};

# 录后想法

【不是很懂】

class Solution {
public:
    int minDepth(TreeNode* root) {

        if(root == nullptr) return 0;

        // 左
        int letfHeight = minDepth(root->left);
        // 右
        int rightHeight = minDepth(root->right);
        // 中，处理节点

        //非叶子节点情况
        if(root->left == nullptr && root->right != nullptr){
            return 1 + rightHeight;
        }
        if(root->right == nullptr && root->left != nullptr){
            return 1 + letfHeight;
        }

        return 1 + min(letfHeight,rightHeight);
    }
};

# 222. 完全二叉树的节点个数

  • 题目：222. 完全二叉树的节点个数 (opens new window)

  • 讲解：文章讲解 (opens new window)、视频讲解 (opens new window)

  • 要求：理解普通二叉树和完全二叉树怎么求

# 录前想法

层序遍历，记录 size

实现代码如下，已 AC (✔️)

class Solution {
public:
    int countNodes(TreeNode* root) {

        queue<TreeNode*> que;
        int count = 0;

        if(root) que.push(root);

        while(!que.empty()){

            int size = que.size();
            count += size;

            while(size--){
                TreeNode* tmpNode = que.front();
                que.pop();

                if(tmpNode->left) que.push(tmpNode->left);
                if(tmpNode->right) que.push(tmpNode->right);
            } 
        }

        return count;
    }
};

# 录后想法

  1. 先求左孩子的数量
  2. 再求右孩子的数量
  3. 加上中节点数量
class Solution {
public:
    int countNodes(TreeNode* root) {

        if(root == nullptr) return 0;

        // 左
        int leftCount = countNodes(root->left);
        // 右
        int rightCount = countNodes(root->right);
        // 中 + 左 + 右
        int all = 1 + leftCount + rightCount;

        return all;
    }
};

# 总结

  1. 确定遍历的方向，选择合适的遍历顺序
  2. （某个）高度和深度的区别，为什么求深度时使用高度的方法？
    • 深度：节点到根节点的距离， 可以上 -> 下 ，适合前序遍历（深度是往下，挖呀挖呀挖）
    • 高度：节点到叶子节点的距离，只能下 -> 上， 适合后序遍历
    • 同一层的深度相同，但高度不一定相同
    • 根节点的高度就是最大深度

  1. 为什么深度需要前序遍历？

    首先目标是向上，那怎么向上，就需要先知道父节点，先知道父节点的遍历方法是：前序遍历

  2. 为什么高度需要后序遍历？

    首先目标是向下，那怎么向下，就需要先知道（全部）子节点，先知道（全部）子节点的遍历方法是：后序遍历
