# 背包问题 (二维)

  • 讲解：文章讲解 (opens new window)、视频讲解 (opens new window)

# 背包问题是什么？

有 n 件物品和一个最多能背重量为 w 的背包。第 i 件物品的重量是 weight [i]，得到的价值是 value [i] 。每件物品只能用一次，求解将哪些物品装入背包里物品价值总和最大。

例子：

背包最大重量为 4，物品为：

重量 价值
物品 0 1 15
物品 1 3 20
物品 2 4 30

问背包能背的物品最大价值是多少？

# 动规五部曲

  1. 确定 dp 数组以及下标的含义

i ：物品

j ：背包剩余的容量

dp[i][j] ：从下标为 [0-i] 的物品里任意取，放进容量为 j 的背包，价值总和最大

  1. 确定递推公式

有两个方向推出来 dp[i][j]

  • 不放物品 i：由 dp[i - 1][j] 推出，即背包容量为 j，里面不放物品 i 的最大价值

  • 放物品 i： dp[i - 1][j - weight[i]] + value[i]

    j - weight[i] 就是放入物品 i ，所以要减去物品 i 的重量 weight[i]

    减去重量后，加上物品 i 的价值 value[i]

递归公式： dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i])

  1. dp 数组如何初始化

状态转移方程 dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i]) 可以看出 i 是由 i-1 推导出来，那么 i 为 0 的时候就一定要初始化。

背包最大重量为 4，物品为：

重量 价值
物品 0 1 15
物品 1 3 20
物品 2 4 30

当背包容量 j = 0 时，一个物品都不能放，所以都为 0

当物品 i = 0 时，其重量为 1，当背包容量为 0 时不能放，其余大于等于 1 时都可以放入物品 0 的值 15

其余坐标，初始化任何值都可以，因为都会在遍历时覆盖

  1. 确定遍历顺序

首先要理解 dp[i-1][j]和dp[i - 1][j - weight[i]] ，左上角方向（包括正上方向）

那么

  • 先遍历 背包 再遍历 物品

  • 先遍历 物品 再遍历 背包

都是可以的

以下是实例的代码：

void test_2_wei_bag_problem1() {
    
    vector<int> weight = {1, 3, 4}; 	// 物品重量
	vector<int> value = {15, 20, 30};   // 物品价值
    int bagweight = 4; 					// 背包容量
    
    // 二维数组
    vector<vector<int>> dp(weight.size(), vector<int>(bagweight + 1, 0));
    
    // 初始化
    // 直接从可容纳第一个物品重量位置开始，巧妙！
    for (int j = weight[0]; j <= bagweight; j++) {
        dp[0][j] = value[0];
    }
    
    // 循环
    for(int i = 1; i < weight.size(); i++) {  // 遍历物品
        for(int j = 0; j <= bagweight; j++) { // 遍历背包容量
            if (j < weight[i]) // 放不下
                dp[i][j] = dp[i - 1][j];
            else               // 放的下
                dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i]);

        }
    }
    
    // 输出
    cout << dp[weight.size() - 1][bagweight] << endl;
}

# 背包问题 (一维)

  • 讲解：文章讲解 (opens new window)、视频讲解 (opens new window)

提示

核心思路 ：

继续题目：

背包最大重量为 4，物品为：

重量 价值
物品 0 1 15
物品 1 3 20
物品 2 4 30

问背包能背的物品最大价值是多少？

  1. 确定 dp 数组的定义

j ：背包容量

dp[j] ：背包容量为 j 时的最大价值

  1. dp 数组的递推公式

二维 dp 数组的递推公式： dp[i][j] = max(dp[i-1][j],dp[i-1][j - weight[i]] + value[i])

同理，一维的 dp[j] = max(dp[j] , dp[j - weight[i]] + value[i])

为什么是 dp[j] ？

可以理解为还没有放入物品，也就没有消耗容量

后面的 dp[j - weight[i]] + value[i] 就是放入了物品，消耗了容量，同时也加上价值

  1. 一维 dp 数组如何初始化

dp[j] 表示：容量为 j 的背包，所背的物品价值可以最大为 dp [j]，那么 dp [0] 就应该是 0，因为背包容量为 0 所背的物品的最大价值就是 0，也就是 dp[0] = 0

再看一下递归公式： dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);

dp[j] 是由 dp[j] 和 dp[j - weight[i]] + value[i] 推到出来，也就是是否更新 dp[j]

所以 dp[j] 需要初始化为正数的最小值，不然过大会覆盖递推关系式（ dp[j-weight[i]]+value[i] ）

所以 dp[j] 都初始化为 0

  1. 如何遍历一维 dp 数组

结论：

  • 二维 dp 遍历的时候，背包容量是从小到大
  • 一维 dp 遍历的时候，背包容量是从大到小。

为什么呢？

因为倒序遍历是为了保证物品 i 只被放入一次！

举一个例子：物品 0 的重量 weight [0] = 1，价值 value [0] = 15

如果是正序遍历

for( int i = 0; i < weight.size(); i++ ) { // 遍历物品
    for( int j = 0; j <= weight[i]; j++ ) { // 遍历背包容量
        dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);
    }
}

dp[2] = dp[2 - weight[0]] + value[0] = 30

此时 dp [2] 就已经是 30 了，意味着物品 0，被放入了两次，所以不能正序遍历

如果是倒叙遍历

for(int i = 0; i < weight.size(); i++) { // 遍历物品
    for(int j = bagWeight; j >= weight[i]; j--) { // 遍历背包容量
        dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);
    }
}

dp[2] = dp[2 - weight[0]] + value[0] = 15 （ dp 数组已经都初始化为 0 ）

dp[1] = dp[1 - weight[0]] + value[0] = 15

所以从后往前循环，每次取得状态不会和之前取得状态重合，这样每种物品就只取一次了。

那么问题又来了，为什么二维 dp 数组历的时候不用倒序呢？

因为对于二维 dp，dp [i][j] 都是通过上一层即 dp [i - 1][j] 计算而来，本层的 dp [i][j] 并不会被覆盖！

那可不可以先遍历背包容量嵌套遍历物品呢？

不可以！

因为一维 dp 的写法，背包容量一定是要倒序遍历（原因上面已经讲了），如果遍历背包容量放在上一层，那么每个 dp [j] 就只会放入一个物品，即：背包里只放入了一个物品。

最后代码实现：

void test_1_wei_bag_problem() {
    vector<int> weight = {1, 3, 4};
    vector<int> value = {15, 20, 30};
    int bagWeight = 4;

    // 初始化
    vector<int> dp(bagWeight + 1, 0);
    for(int i = 0; i < weight.size(); i++) { // 遍历物品
        for(int j = bagWeight; j >= weight[i]; j--) { // 遍历背包容量
            dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);
        }
    }
    cout << dp[bagWeight] << endl;
}

int main() {
    test_1_wei_bag_problem();
}

# 416. 分割等和子集

  • 题目：416. 分割等和子集 (opens new window)

  • 讲解：文章讲解 (opens new window)、视频讲解 (opens new window)

# 录前想法

回溯算法，深度搜索

代码实现如下，未 AC (❌)

class Solution {
private:
    vector<int> path;
    int len;

    bool backtracking(int target, int startIndex, vector<int>& nums) {
        // 当目标和为0时，表示已找到子集，返回true
        if (target == 0) {
            return true;
        }

        // 递归回溯
        for (int i = startIndex; i < len; ++i) {
            // 剪枝：如果当前数字大于目标和，直接跳过
            if (nums[i] > target) {
                continue;
            }
            // 剪枝：如果当前数字和前一个数字相同，跳过以避免重复计算
            if (i > startIndex && nums[i] == nums[i - 1]) {
                continue;
            }
            path.push_back(nums[i]);
            // 递归调用，减去当前数字并移动startIndex
            if (backtracking(target - nums[i], i + 1, nums)) {
                return true;
            }
            path.pop_back();
        }

        return false;
    }

public:
    bool canPartition(vector<int>& nums) {
        path.clear();
        len = nums.size();
        int sum = 0;

        for (int num : nums) {
            sum += num;
        }

        // 如果数组元素和为奇数，不可能平分成两个相等的子集
        if (sum % 2 != 0) {
            return false;
        }

        // 对数组排序，方便剪枝操作
        sort(nums.begin(), nums.end());

        int target = sum / 2;
        return backtracking(target, 0, nums);
    }
};

# 录后想法

核心思路：抽象成 01 背白问题

这题巧妙的隐藏了一个点，“使得两个子集的元素和相等”

这句话的含义就是， 存在集合元素 = 元素全部相加 / 2 ，这一点非常巧妙

因为都是整数，同时也说明当 元素之和 / 2 为奇数时，必然无结果

如何抽象成为 01 背包问题呢？

理清楚背包容量、物品重量、物品价值

  • 背包容量 = 元素之和 / 2
  • 物品质量 = 元素大小
  • 物品价值 = 元素大小
  1. 确定 dp 数组以及下标的含义

j ：背包容量

dp[j] ：当前背包容量的最大值

那么如果背包容量为 target， dp [target] 就是装满背包之后的重量

所以 当 dp [target] == target 的时候，背包就装满了。

  1. 确定递推公式

01 背包的递推公式为： dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);

  • 物品质量 = 元素大小
  • 物品价值 = 元素大小

状态转移方程： dp[j] = max(dp[j], dp[j - nums[i]] + nums[i]);

  1. dp 数组如何初始化

都初始化为 0，因为后面需要跟新 dp [j]，初始化过大会导致丢失递归式的值

  1. 确定遍历顺序

先物品

后背包，且倒叙

// 开始 01背包
for(int i = 0; i < nums.size(); i++) {
    for(int j = target; j >= nums[i]; j--) { // 每一个元素一定是不可重复放入，所以从大到小遍历
        dp[j] = max(dp[j], dp[j - nums[i]] + nums[i]);
    }
}

最后，实现代码如下：

class Solution {
public:
    bool canPartition(vector<int>& nums) {
        
        // 定义dp数组
        int dp[10001] = {0}; // 初始化

        // 求和
        int sum = 0;
        for(int num : nums) sum += num;

        if(sum % 2 == 1) return false;
        // 背包重量
        int bagweight = sum / 2;

        // 循环遍历
        for(int i = 0; i < nums.size(); ++i){                 // 物品
            for(int j = bagweight; j >= nums[i]; --j) {       // 背包
                dp[j] = max(dp[j], dp[j - nums[i]] + nums[i]);
            }
        }
        
        if(dp[bagweight] == bagweight) return true;

        return false;
    }
};

# 总结

  1. 用二维数组 dp 数组实现 01 背包问题
void test(){
    
    vector<int> weight = {1, 3, 4}; 	// 物品重量
	vector<int> value = {15, 20, 30};   // 物品价值
    int bagweight = 4; 					// 背包容量
    
    // 定义二维数组
    vector<vector<int>> dp(weight.size(),vector<bagweight + 1 , 0 >);
   	
    // 初始化
    for(int j = weight[0]; j <= bagweight; j++){
        dp[0][j] = value[0];
    }
    
    // 循环遍历
    for(int i = 0; i < weight.size(); i++){ // 物品
        for(int j = 0; j <= bagweight; j++){ // 背包
            
            if( j < weight[i]){
                dp[i][j] = dp[i-1][j];
            }else{
                dp[i][j] = max(dp[i-1][j],dp[i-1][j - weight[i]] + value[i]);
            }
            
        }
    }
    
	cout << dp[weight.size()-1][bagweight.size()] << endl;
    
    return; 
}

  1. 两个 for 循环颠倒顺序可以吗？

可以颠倒，因为 dp[i][j] 的值取决于上一层

  1. 用一维 dp 数组实现
void test(){
    
    vector<int> weight = {1, 3, 4}; 	// 物品重量
	vector<int> value  = {15, 20, 30};  // 物品价值

            dp[j] = max(dp[j],dp[j - weight[i]] + value[i]);
        }
    }
    
    // 打印结果
    cout << dp[bagweight] << endl;
}

  1. 两个 for 循环颠倒顺序可以吗？

不可以，颠倒后含义就变成了背包中只放入一个物品

正确的含义是，物品只能放入背包一次

  1. for 循环为什么是从后向前遍历，而不是从前向后遍历？

从后往前遍历的目的是背包只使用一次

  1. 二维 dp 数组中为什么可以不管遍历顺序？

可以不管，因为它使用上一层推导而来，有隔离
