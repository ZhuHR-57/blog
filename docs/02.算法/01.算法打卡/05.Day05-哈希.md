## 242.有效的字母异位词

+ 题目：[242.有效的字母异位词](https://leetcode.cn/problems/valid-anagram/)

+ 讲解：[文章讲解](https://programmercarl.com/0242.%E6%9C%89%E6%95%88%E7%9A%84%E5%AD%97%E6%AF%8D%E5%BC%82%E4%BD%8D%E8%AF%8D.html)、[视频讲解](https://www.bilibili.com/video/BV1YG411p7BA/)

+ 要求：数组用来做哈希表



### 录前想法

​	按要求，数组作为哈希表来记录每个字符，s字符串的字符++，t字符串的字符--，若最后tmp数组中都为0则为有效的字母异位词。

以下代码已AC(:heavy_check_mark:)

```cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        
        if(s.size() != t.size()){return false;}

        // 初始化数组
        int tmp[26] = {0};

        // s.size() == t.size()
        for(int i = 0; i < s.size();++i){
            // 记录s字符串
            tmp[s[i] - 'a']++;
            // 从tmp中删除t字符串字符
            tmp[t[i] - 'a']--;
        }

        // 查看tmp是否全为0
        for(int i = 0; i < 26;++i){
            cout<<tmp[i]<<endl;
            if(tmp[i] != 0){
                return false;
            }
        }

        return true;
 
    }
};
```

- 时间复杂度: O(n)
- 空间复杂度: O(1)

## 349.两个数组的交集

+ 要求：使用数组和set两种解法



### 录前想法

> 数组

1. 定义set数据结构，防止结果重复
2. 定义1000数组
3. 记录nums1并判断nums2是否存在
4. 类型转换并返回

已AC(:heavy_check_mark:)

```cpp
class Solution {
public:
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        // 去重结果集
        unordered_set<int> result;

        // 使用数组
        int tmp[1000] = {0};

        // 记录nums1
        for(int i = 0;i < nums1.size();++i){
            tmp[nums1[i]] = 1;
        }

        // 判断nums2元素是否存在
        for(int i = 0;i < nums2.size();++i){
            if(tmp[nums2[i]] == 1){
                result.insert(nums2[i]);
            }
        }

        return vector<int>(result.begin(), result.end());;
    }
};
```

> set数据结构

写出来和上面数组差不多，对SET数据结构操作不熟练。



### 录后想法

主要是**迭代器**的使用，AC(:heavy_check_mark:)

```cpp
class Solution {
public:
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        // 去重结果集
        unordered_set<int> result;
        unordered_set<int> tmp(nums1.begin(),nums1.end());

        // 使用SET
        for(int num : nums2){
            // tmp.find(num) 未找到则返回tmp.end()
            if(tmp.find(num) != tmp.end()){
                result.insert(num);
            }
        }

        return vector<int>(result.begin(), result.end());;
    }
};
```

### 总结

+ SET数据结构的**无重复**
+ 迭代器的使用

### 录后想法

+ 会 **无限循环**，那么也就是说**求和的过程中，sum会重复出现，这对解题很重要！**

​	看了卡哥的讲解后发现是思路错了，不是出现sum为个位数时返回false，而是出现了重复的sum

​	根据这个思路可以写出以下代码：AC(:heavy_check_mark:)

```cpp
class Solution {
public:
    bool isHappy(int n) {

        int sum;
        unordered_set<int> tmp;

        while(1){

            // 循环记录平方和
            sum = 0;
            while(n){
                // 取个数数并平方和
                sum += (n%10)*(n%10);
                // 缩短位数（剔除个位）
                n /= 10;
            }

            // 判断是否为1
            if(sum == 1){
                return true;
            }else{
                n = sum;
            }

            // sum曾经出现过
            if(tmp.find(sum) != tmp.end()){
                return false;
            }else{
                tmp.insert(sum);
            }

        }
    }
};
```

### 总结

+ 无限循环 = 会重复出现值

class Solution {

    vector<int> twoSum(vector<int>& nums, int target) {
        
       for(int i = 0; i < nums.size();++i){
            for(int j = i + 1; j < nums.size();++j){

### 录后想法

思路正确，顺便学习以下map用法，AC(:heavy_check_mark:)

```cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        
        // 将nums装入map中
        std::unordered_map <int,int> map;
        for(int i = 0; i < nums.size(); ++i){
            auto iter = map.find(target - nums[i]); 
            if(iter != map.end()){
                // iter->second 下标
                // iter->frist  值
                return {iter->second, i};
            }
            map.insert(pair<int, int>(nums[i], i)); 
        }

        return {};
    }
};
```

### 总结

+ 判断key有序无序选择使用MAP
